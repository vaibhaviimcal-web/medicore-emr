# Download HTML Prototype

The complete HTML prototype file (`medicore-prototype.html`) is too large to include directly in the repository.

## How to Get the Prototype

### Option 1: Copy from Documentation
The complete HTML code was provided in the project documentation. You can:
1. Find the HTML code in the project messages/documentation
2. Copy the entire HTML content
3. Create a file named `medicore-prototype.html`
4. Paste the content
5. Open in Chrome

### Option 2: Request from Development Team
Contact the development team to receive the complete HTML prototype file.

## What the Prototype Includes

✅ Complete working UI for all features
✅ Patient registration and search
✅ OPD case sheet with quick-select options
✅ E-prescribing with medicine database
✅ Billing and payment tracking
✅ iPad-optimized responsive design
✅ Touch-friendly interface

## File Size
Approximately 50KB (single HTML file with embedded CSS and JavaScript)

## Browser Compatibility
- Chrome (Recommended)
- Firefox
- Safari
- Edge

## Testing Instructions

Once you have the file:
1. Open in Chrome
2. Press F12 for DevTools
3. Toggle Device Toolbar (Ctrl+Shift+M)
4. Select iPad Pro
5. Rotate to Landscape
6. Test all features!

## Note
This is a frontend-only prototype. Data is stored in memory and will be lost on page refresh. For persistent data, use the full-stack version with backend and database.

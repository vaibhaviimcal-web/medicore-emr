# MediCore EMR - HTML Prototype

**Instant demo - No installation required!**

## Quick Start

1. **Download** `medicore-prototype.html`
2. **Open** in any web browser (Chrome recommended)
3. **Test** immediately - no setup needed!

## Testing on iPad Simulator

### Chrome DevTools (Recommended):
1. Open `medicore-prototype.html` in Chrome
2. Press `F12` (or `Ctrl+Shift+I`)
3. Click **Toggle Device Toolbar** icon (or `Ctrl+Shift+M`)
4. Select **"iPad Pro"** from device dropdown
5. Click **Rotate** icon to switch to **Landscape** mode
6. Start testing!

### Firefox Responsive Design:
1. Press `Ctrl+Shift+M`
2. Select iPad dimensions (1366 x 1024)
3. Test features

## Features Included

✅ **Patient Registration** - Add new patients with demographics
✅ **Patient Search** - Search by name, phone, or patient ID
✅ **Dashboard** - View stats and recent patients
✅ **OPD Case Sheet** - Quick-select complaints, diagnosis, vitals
✅ **E-Prescribing** - Quick-select medicines, add custom medicines
✅ **Billing** - Calculate fees, record payments
✅ **iPad-Optimized UI** - Large buttons, touch-friendly interface
✅ **Responsive Design** - Works on all iPad sizes

## What Works

- All UI interactions
- Patient management (in-memory)
- Case sheet creation
- Prescription building
- Billing calculations

## Limitations

⚠️ **Data is NOT persistent** - Refreshing the page will reset all data
⚠️ **No backend** - This is a frontend-only demo
⚠️ **No PDF generation** - Simulated only
⚠️ **No WhatsApp integration** - Simulated only
